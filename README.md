# Agentic Econ Lab Newsletter

A repository for managing and archiving the Agentic Econ Lab newsletter issues.

## About

This repository contains the source content, templates, and archives for the Agentic Econ Lab newsletter. Each newsletter issue is tracked as a GitHub issue and archived in this repository.

## Structure

```
.
├── README.md
├── issues/              # Newsletter issue archives
│   └── templates/       # Newsletter templates
├── .github/
│   └── ISSUE_TEMPLATE/  # GitHub issue templates
└── LICENSE
```

## Newsletter Issues

Newsletter issues are managed as GitHub issues in this repository. Each issue represents a newsletter edition and contains:

- Content outline
- Featured articles
- Research highlights
- Community updates
- Upcoming events
- Resources and tools

## Contributing

### Submitting Content

To suggest content for an upcoming newsletter:

1. Open a new issue using the "Newsletter Content Suggestion" template
2. Fill in the relevant sections
3. Tag it with appropriate labels (e.g., `research`, `tools`, `community`)

### Creating a Newsletter Issue

1. Use the "Newsletter Issue" template to create a new issue
2. Fill in all sections with the planned content
3. Once published, the issue will be archived in the `issues/` directory

## Archive

Published newsletter issues are archived in the `issues/` directory, organized by year and issue number.

## Subscription

To subscribe to the newsletter, visit [agenticecon.org](https://agenticecon.org) or follow this repository for updates.
